# Security Information and Event Management (SIEM)

## Overview

Security Information and Event Management (SIEM) is a security solution that provides centralized collection, normalization, correlation, and analysis of security events generated across an organization's infrastructure. It enables security teams to monitor systems continuously, detect suspicious activities, investigate security incidents, and support compliance requirements.

SIEM platforms collect logs from multiple sources such as operating systems, applications, network devices, firewalls, and endpoints, providing a centralized view of an organization's security posture.

---

## Objectives

The primary objectives of a SIEM are:

* Centralize security log collection.
* Monitor security events in real time.
* Detect suspicious and malicious activities.
* Correlate events from multiple log sources.
* Generate alerts based on predefined detection rules.
* Support incident investigation and response.
* Maintain security logs for auditing and forensic analysis.

---

## Core Functions

A SIEM platform performs the following core functions:

* Log Collection
* Log Normalization
* Event Correlation
* Alert Generation
* Security Monitoring
* Log Analysis
* Incident Investigation
* Reporting and Dashboard Visualization
