# SIEM Lab with Wazuh

## Overview

This repository documents my journey of building a Security Information and Event Management (SIEM) Lab using Wazuh. The primary objective of this project is to gain practical experience with enterprise security monitoring, centralized log management, event correlation, and alert analysis.

The project focuses on understanding how SIEM platforms collect security logs from multiple sources, analyze events, generate alerts, and provide visibility into the security posture of an organization's infrastructure.

This repository serves as both a learning resource and a practical implementation guide for building and managing a SIEM environment using open-source technologies.

---

# Objectives

The primary objectives of this project are:

* Understand the fundamentals of Security Information and Event Management (SIEM).
* Build a functional Wazuh SIEM environment.
* Configure Windows and Linux endpoints.
* Collect and analyze security logs.
* Understand event correlation and alert generation.
* Explore Wazuh dashboards and monitoring capabilities.
* Study security events generated by operating systems.
* Develop practical log analysis skills.
* Document the complete setup and configuration process.
